# 📄 文档处理 (Document Processing)

本类别包含所有与文档创建、编辑、处理和分析相关的技能。

## 技能列表

| 技能名称 | 英文名 | 描述 | 来源 |
|----------|--------|------|------|
| Word文档处理 | docx | 创建、编辑、分析 Word 文档，支持修订追踪、评论和格式设置 | anthropics/skills |
| PDF处理 | pdf | 提取文本、表格、元数据，合并和标注 PDF | anthropics/skills |
| Excel表格处理 | xlsx | 电子表格操作：公式、图表、数据转换 | anthropics/skills |
| PPT演示文稿 | pptx | 读取、生成和调整幻灯片、布局、模板 | anthropics/skills |
| Markdown转EPUB | Markdown to EPUB Converter | 将 Markdown 文档和聊天记录转换为专业 EPUB 电子书 | ComposioHQ |
| 文档格式转换 | doc-coauthoring | 协作文档处理和格式转换 | skills.sh |

## 使用场景

### 1. 文档自动化处理
- 自动批量处理合同文档
- 标准化报告模板生成
- 文档格式统一转换

### 2. 内容提取与分析
- 从 PDF 提取关键信息
- 表格数据提取和转换
- 文档元数据分析

### 3. 电子书制作
- 将 Markdown 笔记转换为 EPUB 格式
- 会议记录整理成电子书
- 知识库文档打包

---

## 安装使用

```bash
/plugin marketplace add docx
/plugin marketplace add pdf
/plugin marketplace add xlsx
/plugin marketplace add pptx
```
