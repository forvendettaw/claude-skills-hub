## 使用场景 (Use Cases)

### 场景 1: 云端 Web 应用部署
- **描述**: 将 Web 应用部署到云平台
- **适用技能**: vercel-react-best-practices, docker-expert, github-actions-templates
- **使用步骤**:
  1. 使用 vercel-react-best-practices 开发应用
  2. 使用 docker-expert 容器化
  3. 使用 github-actions-templates 配置 CI/CD

### 场景 2: Azure AI 服务集成
- **描述**: 在应用中集成 Azure AI 能力
- **适用技能**: azure-ai, azure-observability
- **使用步骤**:
  1. 使用 azure-ai 集成 AI 服务
  2. 使用 azure-observability 监控

### 场景 3: 成本优化
- **描述**: 优化云资源使用成本
- **适用技能**: aws-cost-optimization, azure-cost-optimization
- **使用步骤**:
  1. 分析当前资源使用
  2. 应用成本优化建议
  3. 监控优化效果
