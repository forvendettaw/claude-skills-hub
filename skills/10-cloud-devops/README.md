## 使用场景

### 1. 云基础设施
- AWS/Azure/GCP 资源管理
- 无服务器架构
- 容器化部署

### 2. CI/CD 流水线
- GitHub Actions 工作流
- 自动部署
- 持续集成

### 3. 成本优化
- 云资源成本分析
- 优化建议
- 预算管理

### 4. 监控运维
- 应用性能监控
- 错误追踪
- 告警通知

```bash
/plugin marketplace add aws-skills

/plugin marketplace add vercel-react-best-practices
```
