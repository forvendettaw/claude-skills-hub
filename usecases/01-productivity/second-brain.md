# 第二大脑 (Second Brain)

## 案例概述

构建个人知识管理系统，存储和检索学习内容、记忆和重要信息。

## 核心功能

- 长期记忆存储
- 语义搜索检索
- 知识图谱构建
- 自动关联相关概念

## 技术架构

1. **记忆存储层**：保存重要对话、想法、决策
2. **语义索引层**：使用向量数据库实现语义搜索
3. **检索增强层**：RAG 模式辅助知识调用
4. **呈现层**：通过对话接口访问

## 实施要点

- 定义记忆分类（人物、项目、概念、决策）
- 设置自动记忆触发规则
- 定期回顾和整理记忆
- 使用标签和关系连接知识

## 关联技能

| 技能 | 用途 |
|------|------|
| tapestry | 知识网络构建 |
| knowledge-management | 知识管理 |
| semantic-memory-search | 语义记忆搜索 |
| remember-conversations | 记住对话内容 |

## 使用场景

- 记住人际关系细节（生日、偏好、承诺）
- 项目决策记录
- 学习笔记整理
- 创意灵感收集
- 重要文档归档
