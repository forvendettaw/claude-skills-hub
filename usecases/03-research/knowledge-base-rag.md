# 知识库 RAG (Knowledge Base RAG)

## 案例概述

构建个人或团队知识库，使用 RAG（检索增强生成）技术实现智能问答。

## 核心功能

- 文档自动向量化
- 语义搜索
- 基于上下文的问答
- 来源追溯

## 技术架构

1. **文档处理**：PDF、Markdown、Notion 等
2. **向量化存储**：Chroma、Pinecone 等
3. **检索层**：语义相似度搜索
4. **生成层**：Claude 基于检索结果生成答案

## 实施步骤

1. 收集和整理文档
2. 配置向量化管道
3. 建立检索接口
4. 集成到工作流程

## 关联技能

| 技能 | 用途 |
|------|------|
| knowledge-management | 知识管理 |
| deep-research | 深度研究 |
| article-extractor | 文章提取 |
| web-search | 网络搜索 |

## 使用场景

- 公司内部知识库
- 产品文档问答
- 研究资料管理
- 个人学习笔记
- 客服知识库
