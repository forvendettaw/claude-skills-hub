# 多源科技新闻摘要 (Multi-Source Tech News Digest)

## 案例概述

从 100+ 来源自动聚合科技新闻，生成个性化每日简报。

## 功能特点

- 多源新闻聚合
- AI 摘要生成
- 主题过滤
- 个性化推荐

## 技术要求

- 新闻 API 集成
- 内容提取能力
- 摘要生成
- 定时任务

## 来源类型

- 科技媒体 (TechCrunch, Wired, Verge 等)
- 开发者社区 (Hacker News, Reddit, Dev.to)
- 官方博客 (各科技公司官方博客)
- 聚合平台 (Google News, Bing News)

## 关联技能

| 技能 | 用途 |
|------|------|
| web-search | 新闻搜索 |
| article-extractor | 文章提取 |
| deep-research | 深度研究 |
| copywriting | 摘要撰写 |

## 输出格式

- 简短摘要
- 分类整理
- 原文链接
- 阅读建议
