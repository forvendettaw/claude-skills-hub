# 🤝 贡献指南

欢迎来到 Awesome OpenClaw 项目！感谢您对本项目的关注和支持。

## 如何贡献

### 添加新技能

1. **Fork 本仓库**
2. **创建新分支**: `git checkout -b add-new-skill`
3. **添加技能信息**:
   - 在对应的分类目录下编辑 `README.md`
   - 或创建新的分类目录
4. **提交 Pull Request**

### 技能格式要求

请按以下格式添加技能：

```markdown
| 技能名称 | 英文名 | 描述 | 来源 |
|----------|--------|------|------|
| 示例技能 | example-skill | 技能描述 | 来源 |
```

### 添加使用案例

在 `usecases/` 目录下创建新的使用案例文件：

```markdown
# 使用案例名称

## 场景描述
描述这个使用案例解决的问题

## 使用步骤
1. 步骤一
2. 步骤二

## 预期效果
描述预期的输出结果
```

## 贡献准则

- ✅ 欢迎添加新的 Skills
- ✅ 欢迎补充使用案例
- ✅ 欢迎改进翻译
- ✅ 欢迎修复错误

- ❌ 请勿添加广告内容
- ❌ 请勿添加恶意或有害内容
- ❌ 请勿添加未经授权的破解工具

## 审核标准

提交的内容将根据以下标准审核：

1. **相关性** - 是否与 OpenClaw/Claude Code 相关
2. **实用性** - 是否有实际使用价值
3. **质量** - 描述是否清晰准确
4. **原创性** - 是否为原创内容或已获授权

## 联系方式

如有问题，请提交 Issue 或联系维护者。
